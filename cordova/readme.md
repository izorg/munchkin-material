# Munchkin Cordova

Consume purchase

```javascript
store.inappbilling.consumePurchase(console.log, console.log, "full_version");
```

List iOS emulators

```shell script
cordova emulate ios --list
```

```shell script
rm /usr/local/bin/node && ln -s $(eval which node) /usr/local/bin/node
```
